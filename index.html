Proyecto RogueLike.

  1. Un juego simple pero divertido.
  2. Con estilo visual suave y relajante a la vista.
  3. Acción simple, sin sentimiento de frustración o perdida
  4. Mecanicas simples con poca complejidad pero algo desafiantes.

Se busca crear un juego tipo RougueLike con ambiente tranquilizante para crear un entorno suave y relajante al jugador.
